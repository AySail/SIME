SIME

Style Interactive Markdown Extension

What is SIME?

SIME enhances GitHub Markdown by adding custom styles, interactive buttons, media embedding, and fade effects — all with simple syntax.

Key Features

  • Global and class-based text styling with hex colors (no # needed)
  • Clickable buttons with customizable hover and fade effects
  • Embed images, audio, and video directly in markdown files
  • Interactive fade-in effects for text and elements
  • Simple, intuitive syntax wrapped in carets (^)
